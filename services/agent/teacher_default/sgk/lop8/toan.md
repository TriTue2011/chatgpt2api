# Toán lớp 8 (THCS)

## Đa thức
Nhân chia đa thức. Hằng đẳng thức đáng nhớ.

## Phân thức đại số
Rút gọn, cộng trừ nhân chia phân thức.

## Phương trình bậc nhất một ẩn
ax + b = 0. Phương trình đưa về bậc nhất.

## Hình học
Tứ giác, hình thang, hình bình hành, hình thoi.

## Định lý Thales / đồng dạng (cơ bản)
Tỉ lệ đoạn thẳng. Tam giác đồng dạng.
