# Toán lớp 4 (gợi ý dạy)

## Số tự nhiên lớn
Đọc viết số đến hàng triệu. So sánh và sắp xếp số.

## Phép tính với số lớn
Cộng trừ nhân theo cột. Ước lượng kết quả trước khi tính.

## Phân số
Tử số, mẫu số. Cộng trừ phân số cùng mẫu. So sánh phân số cùng mẫu.

## Diện tích
Diện tích hình chữ nhật bằng chiều dài nhân chiều rộng. Đơn vị xăng-ti-mét vuông, mét vuông.

## Thời gian
Một giờ bằng sáu mươi phút. Bài toán khoảng thời gian.
